ReproZip

Site: http://vida-nyu.github.io/reprozip/

Source: https://github.com/ViDA-NYU/reprozip

ReproZip is a tool aimed at simplifying the process of creating reproducible experiments from command-line executions. It tracks operating system calls and creates a package that contains all the binaries, files, and dependencies required to run a given command on the authorâ€™s computational environment. A reviewer can then extract the experiment in his own environment to reproduce the results, even if the environment has a different operating system from the original one.
