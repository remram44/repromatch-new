Dexy

Site: http://www.dexy.it/

Source: https://github.com/dexy/dexy

Dexy is a literate documentation tool for writing technical documents incorporating source code: users write workflows using YAML-based configuration files, which can blend different programming languages, and Dexy can then execute these workflows and use a templating system where code and dynamic elements are incorporated into documents (e.g., PDF, HTML, and LaTeX).
