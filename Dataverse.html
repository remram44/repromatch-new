Dataverse

Site: http://dataverse.org/

Source: https://github.com/IQSS/dataverse

The Dataverse Network is a repository for sharing, citing, and archiving all research data. It offers a central infrastructure to upload data in all file types (e.g., datasets, documentation, and code), also supporting backups, recovery, data discovery and cataloging, metadata extraction, and preservation.
