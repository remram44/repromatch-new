ActivePapers

Site: https://bitbucket.org/khinsen/active_papers_py/wiki/Home

Source: https://bitbucket.org/khinsen/active_papers_py

ActivePapers is a framework that maintains all the software and datasets of an experiment in a single file, described in a HDF5 document. The tool supports code written in Python and collects all the dependencies between datasets and code modules.
