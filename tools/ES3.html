ES3

Site: http://people.eri.ucsb.edu/~es3/public/new/htdocs/projects/proj-essw.htm

The ES3 (Earth System Science Server) project uses strace to monitor and gather operating system calls, constructing provenance graphs that describe the derivation of data products and that resemble workflow specifications. These graphs can then be queried to discover relevant information regarding the data objects.
