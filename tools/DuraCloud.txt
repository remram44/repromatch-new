DuraCloud is a service from DuraSpace that archives and preserves data in the cloud. It is an application that front-ends commercial and academic cloud providers (e.g., Amazon Web Services) and that provides additional functionalities, such as file health checkup, online data sharing, and ability to move content between different cloud providers.
